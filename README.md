# Student-Portal-website-
This document is a project report on developing a Student Portal web application using Spring Boot and Thymeleaf. The student portal will display students' academic performance data like marks in exams, attendance, and other notices. This will help students track their performance and know where they need improvement. It will also allow students and parents to monitor attendance. The project aims to develop an efficient digital platform to manage student data and reduce unnecessary paperwork for institutions. It describes the objectives, methodology, features, tools and technologies used like Spring Boot, Thymeleaf, HTML, CSS and JavaScript. Screenshots of the backend and frontend development phases are also included.
